BEM-VINDO a  PlacentalGrowth

Uma ferramenta interativa para analisar padrões de crescimento placentário.

PlacentalGrowth é um software fácil de usar e de código aberto projetado para a análise rápida de padrões de crescimento placentário. Ele capacita os profissionais de saúde a explorar e comunicar tendências, desempenhos e resultados críticos relacionados ao crescimento placentário.

 O que pode fazer com PlacentalGrowth?

  • Analisar dados de crescimento placentário para diferentes pacientes e gestações.
  • Visualizar e comparar medidas placentárias ao longo do tempo.
  • Explorar interativamente as curvas de crescimento.
  • Visualizar as informações acerca dos dados submetidos.
  • Calcular rácios do Peso Fetal vs Peso Placentar.
  • Gerar relatórios em PDF resumindo a análise do crescimento placentário.

Se precisar de ajuda, clique no icon ?  no cabeçalho para assistência ou consulte a página Ajuda.
